# Инструкция по созданию релиза на GitHub

## Автоматически через gh (если установлен):
```bash
gh release create v1.0.1 --title "v1.0.1" --notes "Релиз от 23.03.2026"
gh release upload v1.0.1 kiosk-v1.0.1.zip --clobber
```

## Вручную через веб-интерфейс:

1. Перейдите на https://github.com/YOUR_USERNAME/kiosk-updates/releases/new

2. В поле "Tag version" введите: `v1.0.1`

3. В поле "Release title" введите: `v1.0.1 - Парсинг афиши, таймаут, слайд-шоу, автозапуск`

4. В описании укажите:
```
## Новые функции:
- Парсинг афиши с сайта xn--b1admgmggbb7a6b.xn--p1ai
- Переключатель парсинга в админ-панели
- Таймаут возврата на главную при бездействии (30-600 сек)
- Слайд-шоу: фото по интервалу, видео до конца
- Автозапуск приложения при загрузке Windows
- Остановка таймера слайд-шоу при открытии экранов
- Кэширование изображений с сайта
- Детальный просмотр афиши только с картинкой

## Исправления:
- Исключена Яндекс Метрика из парсинга
- Видео в галерее зациклено
- Выход из видео по тапу
- Принудительное закрытие приложения из админ-панели
```

5. Прикрепите файл `kiosk-v1.0.1.zip` (34 МБ) в поле "Attach binaries"

6. Нажмите "Publish release"

---

## Обновление на киоске:

### Вариант 1: Через приложение (если есть функция обновления)
1. Открыть админ-панель
2. Перейти в Настройки
3. Нажать "Обновить"
4. Приложение скачает и установит новую версию

### Вариант 2: Вручную через PowerShell
1. Скачать `update.ps1` и `kiosk-v1.0.1.zip` на киоск
2. Запустить PowerShell от имени администратора
3. Выполнить: `.\update.ps1`

### Вариант 3: Полная переустановка
1. Скачать `kiosk-v1.0.1.zip`
2. Распаковать в `C:\kiosk-updates\` с заменой файлов
3. Перезапустить приложение
